# YAPMS Project - Claude Instructions

This file contains project-level instructions and coding standards for Claude Code to follow when working on this project.

## Project Overview

YAPMS (Yet Another Project Management Software) - A Python application built with PySide6 for basic project management.

## Code Style Guidelines

For detailed coding standards, see:
- @.claude/rules/python-style.md - Python formatting and naming conventions
- @.claude/rules/best-practices.md - General coding best practices

## Core Principles

- Write clear, maintainable code
- Follow existing patterns in the codebase
- Keep functions focused and single-purpose
- Test changes before committing

## Technology Stack

- **Language**: Python 3.x
- **GUI Framework**: PySide6 (Qt for Python)
- **Environment**: Windows development environment

## Getting Started

When working on this project:
1. Activate the virtual environment before making changes
2. Follow the naming conventions in `.claude/rules/python-style.md`
3. Keep code simple and avoid over-engineering
4. Ask for clarification when requirements are ambiguous
